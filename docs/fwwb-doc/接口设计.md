# 智能小车物联网控制系统接口设计文档

## 1. 系统接口概述

### 1.1 接口架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        客户端层                                  │
│    微信小程序 (Vant Weapp)          HAP应用 (ArkUI)              │
│         UDP :8888                        UDP :8888              │
└─────────────────────────────┬───────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                     Python后台服务 (端口8888)                     │
│  TCPServer │ UDPMiniAppService │ DataService                   │
│  UDPService (连接小车 :7788)                                      │
└─────────────────────────────┬───────────────────────────────────┘
                              │ UDP :7788
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                     Hi3861网关 (OpenHarmony)                     │
│         UDP接收任务 │ UDP发送任务 │ UART通信任务                   │
└─────────────────────────────┬───────────────────────────────────┘
                              │ UART2 JSON :115200
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                     STM32电机驱动板                               │
│              PID控制 │ 编码器反馈 │ 模式管理                       │
└─────────────────────────────┬───────────────────────────────────┘
                              │ UART2 Binary :115200
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                     STM32传感器板                                │
│           超声波测距 │ 7路巡线 │ WS2812B RGB                     │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 接口层次划分

| 层次 | 接口类型 | 通信方式 | 协议格式 |
|------|---------|---------|---------|
| 客户端 ↔ 后台 | 应用接口 | TCP/UDP | JSON |
| 后台 ↔ Hi3861 | 设备控制接口 | UDP | JSON |
| Hi3861 ↔ STM32电机板 | 板级接口 | UART | JSON |
| STM32电机板 ↔ 传感器板 | 内部接口 | UART | Binary |

## 2. 客户端 ↔ 后台服务接口

**通信方式**: UDP Socket
**端口**: 8888
**消息格式**: JSON，每条消息以换行符 `\n` 结尾

### 2.1 消息类型总览

| 消息类型 | 方向 | 说明 |
|---------|------|------|
| `ping` | →后台 | 心跳检测 |
| `pong` | ←后台 | 心跳响应 |
| `connect` | →后台 | 连接小车请求 |
| `connect_result` | ←后台 | 连接结果响应 |
| `disconnect` | →后台 | 断开连接请求 |
| `disconnect_result` | ←后台 | 断开连接响应 |
| `control` | →后台 | 控制命令下发 |
| `control_result` | ←后台 | 命令执行结果 |
| `query` | →后台 | 历史数据查询 |
| `query_result` | ←后台 | 查询结果响应 |
| `realtime` | ←后台 | 实时数据推送 |
| `demo_mode` | ↔后台 | 演示模式切换 |
| `smart_light_get` | →后台 | 获取光照状态 |
| `smart_light_status` | ←后台 | 光照状态响应 |
| `smart_light_set_mode` | →后台 | 设置光照模式 |
| `smart_light_set_brightness` | →后台 | 设置光照亮度 |
| `irrigation_calc` | →后台 | 灌溉水量计算 |
| `irrigation_calc_result` | ←后台 | 计算结果响应 |
| `irrigation_execute` | →后台 | 执行灌溉 |
| `irrigation_status` | ←后台 | 灌溉状态推送 |
| `irrigation_query` | →后台 | 灌溉历史查询 |
| `irrigation_query_result` | ←后台 | 历史查询结果 |
| `error` | ←后台 | 错误消息 |

### 2.2 连接管理接口

#### 2.2.1 连接小车

**请求消息**
```json
{
  "type": "connect",
  "carIp": "192.168.1.100",
  "carPort": 7788,
  "deviceId": "car_001"
}
```

| 字段 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| type | string | 是 | 固定为 "connect" |
| carIp | string | 是 | 小车IP地址 |
| carPort | int | 否 | 小车端口，默认7788 |
| deviceId | string | 否 | 设备ID，默认自动生成 |

**响应消息**
```json
{
  "type": "connect_result",
  "success": true,
  "message": "已连接到小车 192.168.1.100:7788",
  "deviceId": "car_001"
}
```

| 字段 | 类型 | 说明 |
|-----|------|------|
| type | string | 固定为 "connect_result" |
| success | bool | 连接是否成功 |
| message | string | 结果描述信息 |
| deviceId | string | 设备ID |

#### 2.2.2 断开连接

**请求消息**
```json
{
  "type": "disconnect"
}
```

**响应消息**
```json
{
  "type": "disconnect_result",
  "success": true,
  "message": "已断开与小车的连接"
}
```

#### 2.2.3 心跳检测

**请求消息**
```json
{
  "type": "ping"
}
```

**响应消息**
```json
{
  "type": "pong",
  "timestamp": 1704067200000
}
```

### 2.3 小车控制接口

#### 2.3.1 控制命令下发

**请求消息**
```json

  }
```

**响应消息**
```json
{
  "type": "control_result",
  "success": true,
  "message": "命令已发送"

```

**command 支持的命令格式**:

| 命令类型 | 命令格式 | 说明 |
|---------|---------|------|
| 电源控制 | `{"carStatus": "on"}` | 开机 |
| 电源控制 | `{"carStatus": "off"}` | 关机 |
| 前进 | `{"carStatus": "run"}` | 向前移动 |
| 后退 | `{"carStatus": "back"}` | 向后移动 |
| 左转 | `{"carStatus": "left"}` | 向左转向 |
| 右转 | `{"carStatus": "right"}` | 向右转向 |
| 停止 | `{"carStatus": "stop"}` | 立即停止 |
| 速度档位 | `{"carSpeed": "low"}` | 低速档 |
| 速度档位 | `{"carSpeed": "middle"}` | 中速档 |
| 速度档位 | `{"carSpeed": "high"}` | 高速档 |
| 运行模式 | `{"carMode": "manual"}` | 手动模式 |
| 运行模式 | `{"carMode": "avoid"}` | 避障模式 |
| 运行模式 | `{"carMode": "line"}` | 巡线模式 |
| 运行模式 | `{"carMode": "path"}` | 路径模式 |
| 摇杆控制 | `{"joyX": 50, "joyY": 100}` | 差速控制 |
| 智能照明 | `{"smartLight": {"mode": "auto", "brightness": 75}}` | 照明控制 |
| 风扇控制 | `{"fan": 1}` | 风扇开关 |

**摇杆控制参数说明**

| 字段 | 类型 | 范围 | 说明 |
|-----|------|-----|------|
| joyX | int | -100 ~ 100 | X轴方向，负值左转，正值右转 |
| joyY | int | -100 ~ 100 | Y轴方向，负值后退，正值前进 |

### 2.4 实时数据接口

#### 2.4.1 实时数据推送（后台 → 客户端）

后台主动推送小车状态数据，推送周期为 50ms。

```json
{
  "type": "realtime",
  "data": {
    "carStatus": "run",
    "carMode": "manual",
    "L_spd": 100,
    "R_spd": 100,
    "carPower": 12000,
    "distance": 300,

  },

}
```

**data 字段说明**

| 字段 | 类型 | 说明 |

| carStatus | string | 小车状态：on/off/run/back/left/right/stop |
| carMode | string | 运行模式：manual/avoid/line/path |
| L_spd | int | 左轮速度 (mm/s) |

| env.temp | int | 温度 (℃) |

### 2.7 智能灌溉接口

#### 2.7.1 灌溉水量计算

**请求**
```json
{
  "type": "irrigation_calc",
  "deviceId": "car_001",
  "data": {
    "temperature": 28,
    "humidity": 55,
    "light": 15000
  }
}
```

**响应**
```json
{
  "type": "irrigation_calc_result",
  "data": {
    "recommendedWater": 680.5,
    "factors": {
      "temperature": 1.06,
      "humidity": 1.075,
      "light": 1.15,
      "history": 1.0,
      "season": 1.1
    },
    "recentIrrigations": [
      {
        "time": "2025-12-28 10:30",
        "amount": 500,
        "status": "completed"
      }
    ]
  }
}
```

**灌溉因子说明**

| 因子 | 计算基准 | 调整规则 |
|-----|---------|---------|
| temperature | 25℃ | 每偏离1°C增加2% |
| humidity | 60% | 每低于1%增加1.5% |
| light | 10000lux | 每增加100%增加10% |
| history | 无历史 | 1小时内灌溉过则减少50% |
| season | 春季基准 | 春1.1/夏1.3/秋1.0/冬0.7 |

#### 2.7.2 执行灌溉

**请求**

| triggerType | string | 触发类型：manual/auto/scheduled |

## 5. STM32电机板 ↔ STM32传感器板接口

**通信方式**: UART2
**波特率**: 115200
**数据格式**: Binary (二进制)

### 5.1 同步包（电机板 → 传感器板）

**帧长度**: 8字节

| 字节索引 | 内容 | 说明 |
|---------|------|------|
| 0 | 0xBB | 帧头 |
| 1-2 | 左轮速度 | 有符号16位整数 |
| 3-4 | 右轮速度 | 有符号16位整数 |
| 5 | 小车状态 | 0=停止, 1=前进, 2=后退, 3=左转, 4=右转 |
| 6 | 运行模式 | 0x00=远程, 0xD0=避障, 0xF0=巡线, 0xE0=路径 |
| 7 | 校验和 | 前7字节异或校验 |

### 5.2 避障数据包（传感器板 → 电机板）

**帧长度**: 4字节

| 字节索引 | 内容 | 说明 |
|---------|------|------|
| 0 | 0xFF | 帧头 |
| 1-2 | 距离值 | 无符号16位整数 (mm) |
| 3 | 校验和 | 前3字节异或校验 |

### 5.3 巡线数据包（传感器板 → 电机板）

**帧长度**: 4字节

| 字节索引 | 内容 | 说明 |
|---------|------|------|
| 0 | 0xAA | 帧头 |
| 1 | 巡线结果 | 7个光电管状态，bit0-6对应各通道 |
| 2 | 模式字节 | 0x00=远程, 0xD0=避障, 0xF0=巡线, 0xE0=路径 |
| 3 | 校验和 | 前3字节异或校验 |

### 6.2 SmartLightService

```python
class SmartLightService:
    def calculate_brightness(self, device_id: str, lux: int,
                           auto_mode: bool = True,
```
