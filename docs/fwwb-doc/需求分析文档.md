# 基于OpenHarmony的智能小车物联网系统需求分析文档

## 1. 项目概述

### 1.1 项目背景

随着物联网技术的快速发展和智慧农业、智慧物流等领域的应用需求不断增长，智能小车作为移动物联网平台的重要载体，集成了环境感知、运动控制、远程通信等多项功能，成为嵌入式开发和物联网教学的典型案例。本项目基于华为OpenHarmony操作系统和Hi3861芯片，设计并实现了一套功能完备的智能小车物联网控制系统。

系统以设施农业大棚为典型应用场景，通过部署多源传感器网络采集土壤、环境参数，结合小车移动平台的灵活性，实现大棚环境巡回监测、智能灌溉指导、农业数据采集等功能。小车平台可搭载不同传感器模块，扩展至环境监测、智能搬运、病虫害巡检等多种应用场景。

### 1.2 项目目标

本项目以Hi3861开发板为核心控制单元，设计一套智能小车物联网控制系统。系统实现以下核心功能：小车运动控制（前进、后退、转向、速度调节）、自动模式控制（避障、巡线、路径规划）、环境数据采集（温度、湿度、光照）、智能照明控制、智能灌溉指导、以及远程监控与数据管理。系统同时提供微信小程序和HAP应用两种客户端，实现对小车的远程控制和数据查看。

### 1.3 系统特色

本系统的特色与创新点体现在以下几个方面：第一，系统采用华为自主研发的OpenHarmony物联网操作系统作为核心软件基座，基于Hi3861芯片实现，为开发者提供完整的嵌入式开发参考；第二，系统采用五层通信架构，从小程序/HAP应用到STM32执行层，协议清晰、层次分明；第三，系统集成了农业环境监测功能，可用于智慧农业场景下的移动巡检和数据采集；第四，系统提供微信小程序和HAP应用双客户端覆盖，HAP应用基于HarmonyOS 6.0.0开发，与OpenHarmony系统实现原生配合。

## 2. 系统总体架构

### 2.1 层次化架构设计

本系统采用层次化架构设计，从下至上分为感知执行层、通信转换层、业务服务层和应用交互层四个层级。

**感知执行层**包含传感器模块和执行器模块。传感器包括超声波传感器（测距避障）、红外巡线传感器（路径识别）、温湿度传感器（环境监测）、光照传感器（光照强度）等。执行器包括电机驱动模块（运动控制）、RGB LED（状态指示）、OLED显示屏（本地显示）等。

**通信转换层**以Hi3861网关为核心，负责协议转换和数据转发。Hi3861通过WiFi与Python后台建立UDP连接，接收应用层的控制指令；同时通过UART接口与STM32电机板通信，实现对小车的控制。

**业务服务层**即Python后台服务，负责数据存储、业务逻辑处理和指令转发。后台服务通过UDP与小车通信获取传感器数据，通过UDP/TCP与客户端应用通信，实现数据的汇聚和分发。

**应用交互层**面向终端用户提供服务入口。系统同时提供微信小程序和HAP应用两套客户端，用户可通过任意终端登录系统查看小车状态、发送控制指令。

### 2.3 硬件平台

**Hi3861开发板**是系统的核心控制单元，基于海思Hi3861芯片，运行OpenHarmony操作系统。Hi3861集成WiFi功能，主频160MHz，具备 достаточные计算能力处理传感器数据和运行轻量级控制算法。

**STM32电机驱动板**负责电机控制和PID闭环调速，采用STM32G030F6Px芯片，支持串口通信接收Hi3861的控制指令，输出PWM信号驱动电机。

**STM32传感器板**负责传感器数据采集，包括超声波测距、7通道红外巡线传感器等，并通过串口向电机板上报传感器数据。

## 3. 功能需求分析

### 3.1 小车控制功能

小车控制功能提供对小车的全方位运动控制能力，支持多种控制方式和速度档位。

#### 3.1.1 基本运动控制

| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 电源控制 | 小车电源开启/关闭 | P0 |
| 前进 | 小车向前移动 | P0 |
| 后退 | 小车向后移动 | P0 |
| 左转 | 小车原地或弧线左转 | P0 |
| 右转 | 小车原地或弧线右转 | P0 |
| 停止 | 立即停止所有电机 | P0 |

#### 3.1.2 速度控制

| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 低速档 | 速度约500mm/s | P1 |
| 中速档 | 速度约800mm/s | P1 |
| 高速档 | 速度约1100mm/s | P1 |
| 摇杆控制 | 模拟量无级变速和差速转向 | P1 |

#### 3.1.3 模式控制

| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 手动模式 | 通过摇杆或按钮直接控制 | P0 |
| 避障模式 | 超声波检测障碍物自动躲避 | P0 |
| 巡线模式 | 沿黑线轨迹自动行驶 | P1 |
| 路径模式 | 按预设路径点序列行驶 | P1 |

### 3.2 自动控制功能

#### 3.2.1 超声波避障

超声波避障功能通过发射超声波并接收回波来检测前方障碍物的距离。当检测到障碍物距离小于阈值（默认300mm）时，小车自动后退并转向，绕过障碍物后继续前进。

| 参数 | 默认值 | 说明 |
|-----|-------|------|
| 避障距离阈值 | 300mm | 触发避障的距离 |
| 后退时间 | 30ms | 后退持续时间 |
| 转向时间 | 30ms | 转向持续时间 |
| 手动优先时间 | 800ms | 手动控制打断自动的窗口 |

#### 3.2.2 红外巡线

红外巡线功能通过检测地面黑线的反射信号实现沿轨迹行驶。传感器板提供7通道ADC，可检测不同位置的黑线偏差，电机板根据偏差计算转向修正量，实现平滑的巡线行驶。

### 3.3 环境监测功能

环境监测功能采集并展示农业大棚内的环境参数，为精准农业管理提供数据支持。

#### 3.3.2 数据展示

### 3.5 智能灌溉功能

智能灌溉功能根据环境参数计算推荐灌溉方案，指导农业生产。

#### 3.5.1 灌溉计算

系统根据温度、湿度、光照等参数，计算推荐灌溉水量：

```
water_amount = base_amount × temp_factor × humi_factor × light_factor × history_factor × season_factor
```

| 因子 | 基准值 | 调整规则 |
|-----|-------|---------|
| temp_factor | 25℃ | 每偏离1°C增加2% |
| humi_factor | 60% | 每低于1%增加1.5% |
| light_factor | 10000lux | 每增加100%增加10% |
| history_factor | 基准 | 1小时内灌溉则减少50% |
| season_factor | 春季基准 | 春1.1/夏1.3/秋1.0/冬0.7 |

#### 3.5.2 灌溉历史

| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 灌溉记录 | 记录每次灌溉的时间和水量 | P2 |
| 历史查询 | 查看历史灌溉记录 | P2 |
