# 智能小车物联网系统 测试报告

---

## 1. 测试概况

### 1.1 项目信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 基于Hi3861 (OpenHarmony) 的智能小车物联网控制系统 |
| 测试版本 | V1.0 |
| 测试周期 | 2026-04-01 至 2026-04-15 |
| 测试类型 | 系统测试、集成测试、验收测试 |
| 编制日期 | 2026-04-19 |

### 1.2 测试目标

验证智能小车物联网控制系统的各项功能是否符合需求规格，包括：
- 小车运动控制（前进/后退/转向/停止）
- 自动避障与巡线功能
- 智能光照调节
- 智能灌溉控制
- 环境数据采集与展示
- 微信小程序与HAP应用双端控制
- 演示模式（模拟数据）

|------|------|

| 严重程度 | 中 |
| 发现日期 | 2026-04-05 |
| 发现用例 | TC-CAR-009 (高速档位) |
| 缺陷描述 | 高速档位实测速度为950mm/s，低于目标值1100mm/s |

| 验证结果 | 修复后速度达1080mm/s，符合±5%误差要求 |
| 验证日期 | 2026-04-06 |

**缺陷 #002: 传感器数据偶发跳变**

| 项目 | 内容 |
|------|------|
| 缺陷ID | DEF-002 |
| 严重程度 | 低 |
| 发现日期 | 2026-04-09 |
| 发现用例 | TC-ERR-007 (传感器数据异常) |
| 缺陷描述 | 温度数据出现偶发性异常跳变（±10℃） |
| 缺陷原因 | 数据滤波窗口偏小(3点)，抗干扰能力不足 |
| 修复措施 | 增加滤波窗口至10个采样点 |
| 验证结果 | 修复后数据稳定性显著改善，连续100次采样无跳变 |
