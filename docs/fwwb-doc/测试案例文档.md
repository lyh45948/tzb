# 智能小车物联网系统 测试案例文档

## 1. 测试概述

### 1.1 项目简介

本项目是基于 Hi3861 (OpenHarmony) 的智能小车物联网系统，实现以下功能：
- 小车运动控制（前进/后退/转向/停止）
- 自动避障与巡线功能
- 智能光照调节
- 智能灌溉控制
- 环境数据采集与展示
- 微信小程序与HAP应用双端控制

### 1.2 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                      客户端层                                │
│         微信小程序 (Vant Weapp)     HAP应用 (ArkUI)        │
│              UDP :8888                   UDP :8888          │
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│               Python后台服务 (端口8888)                      │
│  UDPMiniAppService │ UDPService(连接小车:7788) │ DataService │
└────────────────────────────┬────────────────────────────────┘
                             │ UDP :7788
                             ▼
┌─────────────────────────────────────────────────────────────┐
│                   Hi3861网关 (OpenHarmony)                   │
│        UDP接收任务 │ UDP发送任务(50ms) │ UART通信任务        │
└────────────────────────────┬────────────────────────────────┘
                             │ UART2 JSON :115200
                             ▼
┌─────────────────────────────────────────────────────────────┐
│                     STM32电机驱动板                          │
│               PID控制 │ 编码器反馈 │ 模式管理                │
└────────────────────────────┬────────────────────────────────┘
                             │ UART2 Binary :115200
                             ▼
┌─────────────────────────────────────────────────────────────┐
│                      STM32传感器板                           │
│            超声波测距 │ 7路巡线 │ WS2812B RGB              │
└─────────────────────────────────────────────────────────────┘
```

| 模块 | 测试内容 |
|------|----------|
| 小车控制 | 前进、后退、左转、右转、停止、速度档位、摇杆控制 |
| 自动控制 | 避障模式、巡线模式、路径模式 |
| 智能照明 | 手动/自动模式、亮度调节、时间策略 |
| 智能灌溉 | 水量计算、灌溉执行、历史记录 |
