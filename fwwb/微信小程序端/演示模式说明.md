# 微信小程序演示模式说明

## 更新日期
2025年1月25日

---

## 功能概述

演示模式允许用户在不连接设备的情况下预览小程序的所有功能，包括实时图表展示、模拟数据更新等。这对于产品展示、用户体验测试和功能演示非常有用。

---

## 支持演示模式的页面

### 1. 环境监测页面 (`environment`)

#### 演示内容
- **温度/湿度图表**：预加载150个历史数据点，模拟约5分钟的监测数据
- **传感器数值**：初始显示温度 26.5℃、湿度 65.3%、光照 850Lx
- **RGB氛围灯**：默认颜色 (128, 200, 150) - 柔和的蓝绿色
- **实时更新**：开启演示模式后，每2秒自动更新模拟数据

#### 演示模式功能
- 自动生成正弦波形式的温度和湿度变化
- 光照强度在 700-1000Lx 之间随机波动
- 保持UI交互功能（开关、滑块等）

#### 数据生成逻辑
```javascript
// 温度变化：从24度开始，缓慢上升到28度，再缓慢下降
const tempBase = 24 + Math.sin(progress * Math.PI * 4) * 2;

// 湿度变化：从60%开始，与温度相反
const humiBase = 60 + Math.cos(progress * Math.PI * 3) * 15;
```

### 2. 小车控制页面 (`control`)

#### 演示内容
- **速度曲线图**：预加载50个数据点，展示小车的运动历史
  - 左轮速度：绿色曲线
  - 右轮速度：蓝色曲线
- **电量显示**：默认 85%
- **速度仪表**：显示当前速度 800 mm/s
- **档位显示**：默认中档
- **障碍物距离**：默认 120mm

#### 演示模式功能
- **实时数据更新**：每100ms更新一次速度曲线
- **周期性运动**：模拟不同阶段的运动状态
  - 静止阶段 (0-10点)
  - 前进加速 (10-25点)
  - 稳定前进 (25-35点)
  - 转向阶段 (35-40点)
  - 减速停止 (40-50点)
- **电池消耗**：随机缓慢消耗电量
- **障碍物变化**：模拟障碍物距离在 50-200mm 之间波动

#### 数据生成逻辑
```javascript
// 使用正弦波模拟周期性运动
const baseSpeed = 700 + Math.sin(time * 2) * 300;
const noise = (Math.random() - 0.5) * 100;

const l_spd = Math.round(baseSpeed + noise);
const r_spd = Math.round(baseSpeed * 1.05 + noise);
```

### 3. 路径规划页面 (`path_planning`)

#### 演示内容
- **画布预加载**：页面加载时自动生成圆形路径
- **图形选择**：支持圆形、正方形、三角形、8字形
- **路径绘制**：在画布上清晰显示路径轨迹

#### 演示模式功能
- **自动切换**：每4秒自动切换不同形状
  - 圆形
  - 正方形
  - 三角形
  - 8字形
- **连续演示**：循环展示不同路径形状

## UI 组件

### 演示模式开关

所有支持演示模式的页面顶部都有统一的演示模式开关：

```xml
<view class="demo-mode-bar">
  <van-icon name="play-circle-o" size="16px" color="#1989fa" />
  <text class="demo-mode-text">演示模式</text>
  <van-switch 
    checked="{{ isDemoMode }}" 
    bind:change="toggleDemoMode" 
    color="#1989fa" 
    size="20px"
  />
</view>
```

#### 样式特点
- 渐变背景（蓝色系）
- 左侧有蓝色边框标识
- 播放图标 + 文字说明

## 技术实现

### 数据预加载
- 页面 `onLoad` 时调用 `loadDemoData()`
- 生成初始模拟数据
- 延迟500ms后绘制图表（确保 canvas 已初始化）

### 实时更新
- 使用 `setInterval` 定时更新数据
- 环境监测：每2秒更新一次
- 小车控制：每100ms更新一次
- 路径规划：每4秒切换一次形状

### 数据清理
- 页面卸载时清除定时器
- 防止内存泄漏

## 注意事项

### 1. 与真实设备的区别
- 演示模式的数据是模拟的，不是真实传感器数据
- 操作指令不会发送到实际设备
- 部分功能可能不响应（如控制按钮）

### 2. 性能考虑
- 演示模式会持续生成和更新数据
- 长时间运行可能会消耗一定的系统资源
- 建议演示结束后关闭演示模式

### 3. 数据范围
- 温度：22-28℃
- 湿度：45-75%
- 光照：700-1000Lx
- 速度：0-1200 mm/s
- 电量：20-100%
