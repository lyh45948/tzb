# 智能小车语音控制接口协议 (v2.0)

本项目通过 Hi3861 的 `UART1` 接口与外部语音识别模块通信。语音模块作为从设备，根据识别到的词条向 Hi3861 发送单字节指令。

## 1. 通信参数
*   **接口**：UART1 (GPIO 5: RX, GPIO 6: TX)
*   **波特率**：115200
*   **数据位**：8 bit
*   **停止位**：1 bit
*   **校验位**：None

## 2. 指令集定义 (单字节)

| 指令 (HEX) | 功能描述 | 动作说明 |
| :--- | :--- | :--- |
| **0x10** | **开启电机电源** | 底盘上电，准备接收运动指令 |
| **0x11** | **关闭电机电源** | 底盘断电，切断所有动力 |
| **0x20** | **前进** | 切换到手动模式并执行前进 |
| **0x21** | **后退** | 切换到手动模式并执行后退 |
| **0x22** | **左转** | 切换到手动模式并执行左转 |
| **0x23** | **右转** | 切换到手动模式并执行右转 |
| **0x24** | **停止** | **最高优先级**：立即切换为手动模式，强行打断避障、巡线或路径规划并停车 |
| **0x30** | **进入避障模式** | 开启避障算法，自动规避前方障碍 |
| **0x31** | **进入巡线模式** | 开启巡线算法，沿黑线行驶 |
| **0x40** | **走正方形** | 发送一组预设的路径点，让小车跑出正方形 |
| **0x41** | **走三角形** | 发送一组预设的路径点，让小车跑出三角形 |
| **0x50** | **打开风扇** | 开启外设风扇 |
| **0x51** | **关闭风扇** | 关闭外设风扇 |
| **0x60** | **查询温湿度** | 获取环境温湿度并回传给语音模块 |

## 3. 回传协议 (Hi3861 -> 语音模块)
当执行 **0x60** (查询) 时，Hi3861 会发送两个独立的 6 字节数据包：

### 3.1 温度包 (Message ID: 0x01)
*   格式：`AA 55 01 [TempValue] 55 AA`
*   `[TempValue]`：温度整数值 (1 字节)

### 3.2 湿度包 (Message ID: 0x02)
*   格式：`AA 55 02 [HumiValue] 55 AA`
*   `[HumiValue]`：湿度整数值 (1 字节)

**上层配置建议**：
在语音模块配置界面，请将 `temp` 行为的消息编号设为 **1**，`humi` 行为的消息编号设为 **2**。模块会自动根据包中的编号匹配对应的播报词条。

## 4. 路径指令详解
当触发 **0x40** 或 **0x41** 时，Hi3861 会自动向底座发送多条带有 `d` (距离) 和 `a` (角度) 的 JSON 指令。
