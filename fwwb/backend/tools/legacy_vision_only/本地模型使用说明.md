# 本地 LLaVA 模型使用说明

## 方案概述

混合方案（推荐）：
| 任务       | 模型                     | 显存需求 | 说明                    |
|------------|--------------------------|----------|-------------------------|
| 障碍物检测 | YOLOv8                   | 约 2GB   | 已有，实时检测+距离估算 |
| 场景描述   | LLaVA-1.5-7B (4-bit量化) | 约 4-6GB | 本地视觉大模型替代 API  |

## 使用步骤

### 方法一：使用下载脚本（推荐）

1. **运行下载脚本**
   ```bash
   cd d:\挑战杯\1\vision_only
   python download_model.py
   ```

2. **选择镜像站点**
   - 选项 1: hf-mirror（推荐，国内速度快）
   - 选项 2: ModelScope（国内镜像）
   - 选项 3: HuggingFace 官方源（需要网络）

3. **等待下载完成**
   - 模型大小约 10GB
   - 下载时间取决于网络速度

4. **更新配置文件**
   下载完成后，脚本会显示模型路径，修改 `config/config.json`：
   ```json
   {
     "VISION": {
       "USE_LOCAL_MODEL": true,
       "LOCAL_MODEL_NAME": "D:\\挑战杯\\1\\models\\liuhaotian_LLaVA-1.5-7B"
     }
   }
   ```

5. **运行程序**
   ```bash
   cd d:\挑战杯\1
   python esp32_viewer/capture_still.py
   ```
   选择模式 2（本地模式）

### 方法二：手动下载

1. **访问镜像站点**
   - https://hf-mirror.com/liuhaotian/LLaVA-1.5-7B
   - 或 https://www.modelscope.cn/models/AI-ModelScope/liuhaotian/LLaVA-1.5-7B

2. **下载所有文件**
   - 下载所有模型文件（约 10GB）
   - 保存到本地目录，例如：`D:\models\LLaVA-1.5-7B`

3. **更新配置文件**
   ```json
   {
     "VISION": {
       "USE_LOCAL_MODEL": true,
       "LOCAL_MODEL_NAME": "D:\\models\\LLaVA-1.5-7B"
     }
   }
   ```

4. **运行程序**
   ```bash
   cd d:\挑战杯\1
   python esp32_viewer/capture_still.py
   ```
   选择模式 2（本地模式）

## 系统要求

### 硬件要求
- **GPU 显存**: >= 6GB（4-bit 量化模式）
- **磁盘空间**: >= 15GB（模型文件 + 缓存）
- **内存**: >= 8GB

### 软件要求
- Python 3.8+
- CUDA 11.8+（如果使用 GPU）
- 已安装依赖：`pip install -r requirements.txt`

## 常见问题

### 1. 下载失败怎么办？
- 检查网络连接
- 尝试使用 VPN 或代理
- 使用国内镜像站点（hf-mirror 或 ModelScope）
- 分批下载模型文件

### 2. 显存不足怎么办？
- 降低量化精度（修改 `LOAD_IN_4BIT: false`，但需要更多显存）
- 使用 CPU 模式（速度较慢）
- 使用更小的模型（如 LLaVA-1.5-13B 需要更多显存）

### 3. 模型加载很慢？
- 首次加载需要将模型加载到显存，需要几分钟
- 后续使用会更快
- 可以考虑使用 SSD 存储模型文件

### 4. 网络问题导致无法下载？
- 使用 `download_model.py` 脚本，选择国内镜像
- 手动下载模型文件
- 使用离线安装包

## 配置说明

### config.json 配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| USE_LOCAL_MODEL | 是否使用本地模型 | false |
| LOCAL_MODEL_NAME | 模型名称或本地路径 | liuhaotian/LLaVA-1.5-7B |
| LOAD_IN_4BIT | 是否使用 4-bit 量化 | true |

### 本地路径示例
- Windows: `D:\\models\\LLaVA-1.5-7B`
- Linux: `/home/user/models/LLaVA-1.5-7B`

## 性能对比

| 模式 | 优势 | 劣势 |
|------|------|------|
| API 模式 | 无需本地资源，速度快 | 需要网络，有 API 费用 |
| 本地模式 | 无需网络，无 API 费用 | 需要显存，首次加载慢 |
