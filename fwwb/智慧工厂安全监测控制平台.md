# 智慧工厂安全监测控制平台

## 项目概述

本项目是基于国产操作系统（OpenHarmony Hi3861）的智慧工厂安全监测控制平台，参加"挑战杯"揭榜挂帅擂台赛。平台实现工厂环境的实时监测、危险气体报警、AGV避障、货物计数等功能。

## 功能模块

### 1. 温湿度智能监控
- **传感器**: SHT20 (I2C)
- **功能**: 实时监测生产环境温湿度，自动记录数据并可设置阈值报警
- **阈值**: 温度 30℃(警告) / 35℃(危险), 湿度 75%(警告) / 80%(危险)

### 2. 红外感应照明
- **传感器**: AP3216C (环境光/接近) + PIR人体红外感应
- **功能**: 根据光照强度和人流量自动调节照明，实现节能效果
- **应用场景**: 车间照明控制、走廊感应灯

### 3. 危气监测
- **传感器**: SGP30 (CO2/TVOC), MQ-2 (烟雾), MQ-7 (CO)
- **功能**: 监测多种危险气体，超标时触发声光报警并推送通知
- **阈值**:
  - CO2: 800ppm(警告) / 1000ppm(危险)
  - 烟雾: 300(警告) / 500(危险)
  - CO: 35ppm(警告) / 50ppm(危险)

### 4. AGV避障系统
- **传感器**: HC-SR04超声波传感器 (STM32)
- **功能**: 实时监测AGV前方障碍物距离，自动停车或绕行
- **阈值**: 30cm(警告) / 15cm(危险)

### 5. 货物感应计数
- **传感器**: IR红外对管 (E3F-DS30C)
- **功能**: 统计生产线货物通过数量，支持班次统计和累计计数

## 技术特点

### 国产操作系统
- **OpenHarmony LiteOS**: 基于Hi3861芯片，完全国产自主可控
- **响应迅速**: 实时操作系统特性，保证工业场景的低延迟响应

### 通信协议
- **UDP实时通信**: 50ms数据刷新周期，保证监测时效性
- **JSON协议**: 跨平台数据交换，兼容性好

### 数据可视化
- **微信小程序**: 无需安装，即开即用的监控界面
- **Vant Weapp组件**: 专业UI组件库，保证界面美观
- **实时仪表盘**: 圆形仪表盘、数字卡片、趋势图等多种展示方式

### 警报系统
- **多级警报**: 警告(Warning)、危险(Danger)、紧急(Critical)
- **自动控制联动**: 警报触发时可自动开启风扇、警铃等设备
- **历史记录**: 完整的警报记录，支持追溯分析

## 硬件接口

### I2C设备 (GPIO_9/SCL, GPIO_10/SDA)
| 设备 | 地址 | 功能 |
|------|------|------|
| SHT20 | 0x80 | 温湿度传感器 |
| SGP30 | 0x80 | CO2/TVOC传感器 |
| AP3216C | 0x3C | 环境光/接近传感器 |
| PCF8574 | 0x43 | IO扩展器 |

### UART设备
| 接口 | 引脚 | 功能 |
|------|------|------|
| UART1 | GPIO_5(RX), GPIO_6(TX) | 语音模块 |
| UART2 | GPIO_12(RX), GPIO_11(TX) | STM32通信 |

### PCF8574 I/O定义
| 引脚 | 功能 |
|------|------|
| P0 | 风扇控制 |
| P1 | 蜂鸣器 |
| P2 | LED指示灯 |
| P4 | 火焰传感器 |
| P5 | 燃气传感器 |
| P6 | 货物计数脉冲 |
| P7 | PIR人体感应 |

## 数据库表结构

### factory_sensor_data (工厂传感器数据)
实时存储各类传感器监测数据

### alert_records (警报记录)
存储警报历史，支持查询和分析

### equipment_control (设备控制记录)
记录所有设备控制操作，支持追溯
